A LINGUAGEM HTML 5

          HTML (abreviação para a expressão inglesa HyperText Markup
           Language, que significa Linguagem de Marcação de Hipertexto) é uma
           linguagem de marcação utilizada na construção de páginas na web.
           Documentos HTML podem ser interpretados por navegadores.

           Até este momento, vimos os seguintes recursos da linguagem:

           

             

                Recursos 
                Tags associadas 
                Descrição 
             


             

                Links 
                 <a>
                Criar links entre as páginas 
             


             

                 Listas 

                

                 <ul> e <li> 
                 criar lista de itens ordenados 
               


                

                  <ol> e <li> 
                  Criar lista de itens não ordenados  
               



                


                

                   imagens 
                   <img> 
                   inserir imagens
